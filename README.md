## Global node js packages
```bash
$ npm install -g ts-node@10.5.0
$ npm install -g nodemon
```

## Environment
* need create .env file

## Important!
###Before starting:
Install Docker Desktop to your machine and start docker containers:
```bash
$ docker-compose up -d
```

## Start project
```bash
$ npm install
$ npm run start:dev
```
